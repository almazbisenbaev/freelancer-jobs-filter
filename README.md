# How to use

1. Download the repo, unpack
2. Open `script.js`, enable/disable features that you need and don't need, edit keywords and variables to your preference
3. Enable Developer mode in Chrome
4. Click "Load unpacked", select "freelancer-job-filter" folder
5. Go to freelancer.com

---

The extension must remove 90% of jobs that are posted by indian scammers or have too many responses.

---

# Features:

- Hides jobs that contain blacklisted keywords
- Highlights jobs that contain whitelisted keywords
- Hides jobs that have too many bids by other freelancers
- Highlights jobs that have a few bids
